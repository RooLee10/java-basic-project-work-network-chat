/roleslist - список всех ролей
/addrole - добавить роль пользователю (/addrole username rolename)
/removerole - удалить роль у пользователя (/removerole username rolename)
/ban - бан пользователя (/ban username -[1-9] время бана, указывается в часах), (/ban username -* бан навсегда)
/shutdown - остановка сервера
